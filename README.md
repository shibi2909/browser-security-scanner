# Browser Security Scanner (Chrome Extension)

A beginner-friendly Chrome extension that detects insecure password fields,
phishing-style login pages, and warns users in real time.

## Features
- Detects password input fields
- Flags password fields on non-HTTPS pages (High Risk)
- Handles dynamically loaded pages (Facebook, Gmail)
- Shows warnings via extension popup

## Technologies Used
- JavaScript
- Chrome Extension APIs (Manifest V3)
- HTML
